
# Responsive Landing Page (Tailwind CSS)

## 📌 Project Overview

This project is a **responsive landing page** built using **HTML** and **Tailwind CSS**.
It features a modern layout with a navigation bar, hero section, and call-to-action button.

The design adapts to different screen sizes using **Tailwind's responsive utilities**, making it mobile-friendly.

---

[live](https://amrithaamzz.github.io/Responsive-Landing-Page/)
![image](./image.png)

## 🚀 Features

* Responsive navigation bar
* Mobile menu icon
* Hero section with headline and description
* Call-to-action button
* Responsive layout using Flexbox
* Styled using Tailwind CSS CDN

## ▶️ How to Run

1. Download or clone the project.
2. Make sure **image.png** is in the same folder as `index.html`.
3. Open `index.html` in your browser.

No build tools or installation required.

---

## 📱 Responsive Design

The page adjusts layout based on screen size:

* **Mobile:** Stacked layout with menu icon
* **Desktop:** Navigation links visible with side-by-side hero layout

Tailwind responsive classes used:

* `lg:flex`
* `lg:hidden`
* `lg:w-1/2`
* `md:text-6xl`
